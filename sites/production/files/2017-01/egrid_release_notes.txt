The following new files are available on the eGRID web site -- epa.gov/energy/egrid :

  eGRID2014_data.zip contains all files initially released for eGRID2014 including operational data for year 2014:

			eGRID2014_Data.xls
			eGRID2014_Data_Metric.xls
			eGRID2014_GHGOutputRates.pdf
			eGRID2014_SummaryTables.pdf
			eGRID2014_SummaryTables.xls
                     	eGRID2014_eGRID_subregions.jpg
			eGRID2014_NERC_regions.jpg
			eGRID2014_TechnicalSupportDocument.pdf
			eGRID_Release_Notes_2017-01-13.txt (this file)

  eGRID_All_Files_1996_thru_2014_data.zip contains all files from all previous editions of eGRID including operational data for the following years: 1996, 1997, 1998, 1999, 2000, 2004, 2005, 2007, 2009, 2010, and 2012.
